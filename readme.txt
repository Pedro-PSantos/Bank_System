Para executar o programa, são precisos alguns pré-requisitos:


1. Instalação do JDK (Java Development Kit) https://www.oracle.com/pt/java/technologies/downloads/

2. Instalação do Visual Studio Code
https://code.visualstudio.com/

3. Instalação da extensão "Java Extension Pack" no Visual Studio Code: https://marketplace.visualstudio.com/items?itemName=vscjava.vscode-java-pack

4. Instalação do SWI-Prolog
https://www.swi-prolog.org/Download.html

5. Instalação e configuração da biblioteca JPL (Java-Prolog Interface) https://jpl7.org/

Para executar o programa, siga os passos:

1. Abrir o Visual Studio Code.
2. Selecionar as opções "File" > "Open Folder", selecionando a pasta onde se encontra o projeto.
3. Abrir o ficheiro App.java no Visual Studio Code.
4. Clique no botão "Run" na parte superior direita da janela do editor.

!!Importante!!
A biblioteca JPL tem de estar bem configurada, uam vez que pode não ser detetada a sua localização, dando em erro as funções utilizadas desta biblioteca, no programa Java.
